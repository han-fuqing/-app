# 旧时光中古市集

一个可直接部署到 Cloudflare Pages 的移动端中古商品比价 MVP。

## 已有功能

- 商品名称实时搜索
- 当前价格与市场参考价格展示
- Mercari、Yahoo!拍卖、Rakuma 模拟来源
- 价格升序/降序与捡漏推荐排序
- 自动计算 0～100 捡漏评分
- localStorage 本地收藏及收藏列表
- 日语默认、中文切换
- 手机优先的响应式界面

## 本地运行

无需安装依赖。在当前目录执行：

```powershell
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## Cloudflare Pages 部署

这是纯静态站点，无需构建命令。部署时将输出目录设置为 `/`（仓库根目录）即可。
